# WeatherApp-Python 🌤️

تطبيق سطح مكتب بسيط مبني بـ Python و Tkinter بيجيب حالة الطقس لأي مدينة في العالم لحظياً باستخدام OpenWeatherMap API.

## 📌 المميزات
- البحث عن الطقس لأي مدينة بالاسم
- عرض درجة الحرارة، الرطوبة، سرعة الرياح، وحالة السماء
- واجهة رسومية سهلة وبسيطة
- معالجة الأخطاء لو المدينة مش موجودة

## 🛠️ الأدوات المستخدمة
- `Python 3`
- `Tkinter` للواجهة الرسومية
- `Requests` للتعامل مع الـ API
- `OpenWeatherMap API`
