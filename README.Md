# 🌤️ Weather_Project

تطبيق بايثون بيجيب حالة الطقس لاي مدينة في العالم باستخدام OpenWeatherMap API

## ✨ المميزات
- عرض درجة الحرارة، الرطوبة، وسرعة الرياح
- واجهة رسومية بسيطة باستخدام Tkinter
- بيجيب حالة الطقس لحظيا

## 🛠️ التقنيات المستخدمة
- Python
- Requests Library
- Tkinter
- OpenWeatherMap API

## 🚀 طريقة التشغيل
1. نزل المكتبة: `pip install requests`
2. شغل البرنامج: `python main.py`
3. اكتب اسم المدينة وشوف الجو

## 📝 ملاحظة
المشروع ده جزء من تدريبي على الـ APIs
